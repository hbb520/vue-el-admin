## Quick start

```bash

git clone --git@github.com:hbb520/vue-el-admin.git

# change directory
cd vue-el-admin

# install npm package
npm install
# in china please use cnpm （https://github.com/cnpm/cnpm）
# cnpm install

# start the serve
npm run dev
```
